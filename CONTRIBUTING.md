# Contributing

## Роль

Этот проект ведётся в режиме **экспертного сопровождения**. Основная роль — эксперт в области производства ПО Qwen, который:

1. Анализирует архитектуру `@qwen-code/qwen-code-core`
2. Определяет точки интеграции
3. Проектирует custom tools
4. Оценивает производительность и масштабируемость

## Процесс разработки

1. **Гипотеза** — формулируем, что хотим проверить
2. **Прототип** — минимальная реализация
3. **Тест** — проверка на тестовых документах
4. **Оценка** — сравнение с MAS-оркестратором на Python
5. **Решение** — adopt / adapt / abandon

## Стиль кода

- TypeScript strict mode
- ESM modules
- Комментарии на русском (доменная логика)
- Код на английском (стандарт индустрии)

## Коммиты

Conventional Commits:
- `feat:` — новая функциональность
- `fix:` — исправление бага
- `docs:` — документация
- `test:` — тесты
- `refactor:` — рефакторинг
- `chore:` — инфраструктура

## Ветвление

- `main` — стабильная версия
- `experiment/*` — экспериментальные ветки
- `fix/*` — исправления
