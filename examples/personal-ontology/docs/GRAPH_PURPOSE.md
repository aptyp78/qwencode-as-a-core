# Граф Фрадкова П.М. — инструмент стохастического сопоставления

## Назначение

Граф — не архив фактов, а **инструмент сопоставления** при поступлении новых документов в среду AI Canvas.

## Принцип работы

```
Поступление нового документа
  ↓
Извлечение сущностей и связей
  ↓
Сопоставление с графом Фрадкова
  ↓
Стохастическое моделирование:
  • Вероятностные связи
  • Паттерны деятельности
  • Контекстное обогащение
  ↓
Обогащение понимания документа
```

## Что даёт граф

### 1. Контекстное сопоставление

При поступлении документа граф позволяет:
- Определить, связан ли документ с известными видами деятельности Фрадкова
- Выявить пересечения с известными организациями, проектами, событиями
- Обогатить понимание документа через_known контекст

**Пример:**
- Поступает документ о "ИИ-платформе для ОПК"
- Граф показывает: это связано с проектом "ИИ-платформа ОПК" (узел project_2)
- Контекст: Фрадков руководит этим проектом, цель — оркестрация экосистемы ОПК
- Обогащение: документ получает дополнительный смысл через связь с известным проектом

### 2. Стохастическое моделирование

Граф позволяет моделировать вероятностные связи:

**Вероятностные связи:**
- Если документ упоминает "ПСБ" → вероятность связи с "ОПК" (высокая)
- Если документ упоминает "лёгкая атлетика" → вероятность связи с "ВФЛА" (высокая)
- Если документ упоминает "трансформация" → вероятность связи с "Программой 25-25-25" (средняя)

**Паттерны деятельности:**
- Фрадков часто выступает на ПМЭФ → новые выступления на ПМЭФ вероятны
- Фрадков встречался с Путиным по вопросам ГОЗ → новые встречи вероятны
- Фрадков публиковался по теме экспорта → новые публикации вероятны

**Контекстное обогащение:**
- Документ упоминает "санкции" → граф даёт контекст: Фрадков под санкциями с 2022
- Документ упоминает "ВЭБ" → граф даёт контекст: Фрадков работал в ВЭБ (2005-2016)
- Документ упоминает "экспорт" → граф даёт контекст: Фрадков руководил ЭКСАР и РЭЦ

### 3. Выявление противоречий

Граф позволяет выявлять противоречия между новым документом и известным контекстом:

**Пример:**
- Документ утверждает: "ПСБ не участвует в гособоронзаказе"
- Граф показывает: ПСБ — опорный банк ОПК, Фрадков докладывает Путину о ГОЗ
- Противоречие: документ противоречит известному контексту
- Действие: пометить как "требует проверки"

### 4. Обогащение онтологии

Граф позволяет обогащать онтологию среды новыми связями:

**Пример:**
- Поступает документ о "цифровом рубле"
- Граф не содержит прямого узла "цифровой рубль"
- Но граф содержит: "Финансовая инфраструктура", "Цифровая трансформация"
- Обогащение: создать вероятностную связь "цифровой рубль" → "Финансовая инфраструктура"
- Результат: онтология среды обогащается новой связью

## Технические компоненты

### 1. Векторное сопоставление

- Все узлы графа векторизованы (qwen3-embedding:8b, 4096d)
- Новый документ → извлечение сущностей → векторизация
- Поиск ближайших узлов в векторном пространстве (cosine similarity)
- Порог сходства: 0.75 (настраиваемый)

### 2. Графовое сопоставление

- Поиск структурных связей между извлечёнными сущностями и узлами графа
- Анализ путей между узлами (графовый поиск)
- Вычисление "центральности" узлов (важность в графе)

### 3. Стохастическое моделирование

- Вероятностные связи между темами (на основе исторических данных)
- Паттерны деятельности (частота событий, периодичность)
- Байесовское обновление вероятностей при поступлении новых данных

### 4. Противоречия

- Проверка согласованности нового документа с известным контекстом
- Выявление логических противоречий
- Пометка "требует проверки" для противоречивых данных

## Примеры использования

### Пример 1: Поступление документа о ПМЭФ

**Документ:** "Фрадков выступил на ПМЭФ-2026 с докладом о финансовой архитектуре"

**Сопоставление с графом:**
- Узел "ПМЭФ-2026" (event_1) — прямое совпадение
- Узел "Финансовая архитектура" (topic) — векторное сходство 0.82
- Узел "Фрадков П.М." (person_fradkov_pm) — структурная связь

**Результат:**
- Документ связан с известным событием (ПМЭФ-2026)
- Тема соответствует известным интересам Фрадкова
- Контекст обогащён: это не первое выступление на ПМЭФ (были в 2023, 2024, 2025)
- Вероятность связи: 0.95

### Пример 2: Поступление документа о новом проекте

**Документ:** "ПСБ запускает программу поддержки стартапов в области ИИ"

**Сопоставление с графом:**
- Узел "ПСБ" (org_8) — прямое совпадение
- Узел "ИИ-платформа ОПК" (project_2) — векторное сходство 0.78
- Узел "Инновации" (activity) — векторное сходство 0.71

**Результат:**
- Документ связан с известной организацией (ПСБ)
- Тема соответствует известным проектам (ИИ-платформа)
- Контекст обогащён: это продолжение стратегии цифровизации
- Вероятность связи: 0.85
- Рекомендация: проверить связь с проектом "ИИ-платформа ОПК"

### Пример 3: Поступление противоречивого документа

**Документ:** "Фрадков покинул пост председателя ПСБ"

**Сопоставление с графом:**
- Узел "Фрадков П.М." (person_fradkov_pm) — прямое совпадение
- Узел "Председатель Правления ПСБ" (pos_8) — структурная связь
- Дата документа: 2026-07-15

**Результат:**
- Документ противоречит известному контексту (Фрадков — действующий председатель)
- Граф не содержит события "уход с поста"
- Противоречие: документ vs граф
- Действие: пометить как "требует проверки", запросить дополнительные источники
